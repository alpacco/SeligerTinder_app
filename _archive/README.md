# Архив старого Node.js бэкенда

Эта папка содержит старые файлы Node.js бэкенда, которые были заменены на Python версию.

## Структура

- `nodejs_backend/` - все файлы старого Node.js бэкенда
  - `app.js` - основной Express сервер
  - `bot.js` - Telegram бот
  - `db.js`, `db-pg.js` - работа с БД
  - `routes/` - все роуты API
  - `middleware/` - middleware
  - `utils/` - утилиты
  - `package.json` - зависимости Node.js
  - и другие файлы

## Примечание

Все функции из Node.js версии были перенесены в Python бэкенд (`python_backend/`).
Эти файлы сохранены для справки и могут быть удалены в будущем.
